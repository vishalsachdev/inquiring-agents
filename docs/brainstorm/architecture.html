Worker Routes

GET /
Serves the full HTML/CSS/JS app as a single inline response. No static assets needed.
POST /api/debate
Runs one philosopher's turn. Streams SSE back to browser. Body: {philosopher, round, topic, history, injection?}
POST /api/synthesize
Coordinator synthesizes final policy from all positions. Streams SSE.

Key Design Decisions

Single file Worker
HTML is a template string inside the Worker. One wrangler deploy, zero build step.
SSE streaming
OpenAI stream → Worker transforms → SSE to browser. Text appears token-by-token.
Client-side orchestration
Browser drives the sequence (R0→inject?→R1→inject?→R2→synth). Worker is stateless — just proxies to OpenAI.
API key in Worker secret
OPENAI_API_KEY stored as Wrangler secret. Never exposed to browser.
